知识点总结

while True 循环
无限循环，让 AI 可以连续多轮调用工具。只有遇到 return 才退出。
API 调用
client.messages.create() 把对话记录和工具列表发给 Claude，等待回复。
stop_reason 判断
Claude 停止的原因。"tool_use" = 要用工具，继续循环；其他 = 任务完成，退出。
** 字典解包
**block.input 把字典拆成关键字参数传给函数，如 {"city":"北京"} 变成 city="北京"。
工具结果格式
每个结果需要 type、tool_use_id、content 三个字段，Claude 才能对应上。
循环流程
调用 API → 保存回复 → 判断是否要工具 → 执行工具 → 结果发回 → 再次调用 API。
