28 # 3. 最简单的 class —— 造一个"人"
class = 模板/蓝图，用来批量制造同类对象
29 class Person:
class 定义一个类，名字 Person 首字母大写（约定俗成），冒号开启类体
30 def __init__(self, name, age): # 初始化，创建时自动调用
__init__ 是特殊方法，对象被创建时自动执行；self 指代对象自身；name age 是传入参数
31 self.name = name # self.name = 这个人的名字
self.name 把参数 name 存成对象的属性，之后可用 对象.name 访问
32 self.age = age # self.age = 这个人的年龄
同理，self.age 存储年龄属性
33
34 def introduce(self): # 方法 = 这个对象能做的事
def 在 class 内定义方法；第一个参数永远是 self（调用时不用传）
35 print(f"我叫{self.name}，今年{self.age}岁")
方法体：用 self.name 和 self.age 访问对象自己的属性
36
37 def is_adult(self):
另一个方法，会返回一个布尔值
38 return self.age >= 18
return 返回 True 或 False；>= 比较运算符

41 小白 = Person("小白", 25)

42 xiaoming = Person("小明", 15)

44 # 用点号访问属性和方法
造好对象后，用点号使用它
45 print(小白.name) # 属性：小白 的 name

小白.age → 输出 25
47 小白.introduce() # 方法：小白 的 自我介绍()
小白.introduce() → 调用方法，有括号；输出 我叫小白，今年25岁
48 print(f"成年人？{小白.is_adult()}")
方法调用结果直接嵌入 f-string；is_adult() 返回 True
49

54 # 4. 链式点号 —— 一层一层往里找
a.b.c → 先找 a 的 b，再找 b 的 c
55 class Phone:
class Phone → 新的类，表示手机
56 def __init__(self, brand, owner):
参数 brand 品牌，owner 主人（将是另一个对象）
57 self.brand = brand
存储品牌属性
58 self.owner = owner # owner 是一个 Person 对象
self.owner 存的不是字符串，而是一个 Person 对象 → 对象可以嵌套
59
60 my_phone = Phone("iPhone", 小白)
创建 Phone 对象，owner 传入之前创建的 小白 对象
61
62 # 链式访问
链式调用：一层层往里找
63 print(my_phone.brand) # 手机的品牌
一层点号：my_phone.brand → 输出 iPhone
64 print(my_phone.owner.name) # 手机的主人的名字
两层点号：my_phone.owner 得到 Person 对象，再 .name → 小白
65 print(my_phone.owner.age) # 手机的主人的年龄
同理，链式找到 age → 输出 25
66 my_phone.owner.introduce() # 手机的主人.自我介绍()
链式调用方法：my_phone.owner.introduce() → 执行 Person 的 introduce 方法
67 print("---")
