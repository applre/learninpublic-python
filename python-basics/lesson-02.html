▎第一部分：最简单的 if
条件为真就执行缩进块
4 # 1. 最简单的 if
注释：第一个知识点开始
5 age = 18

6
7 if age >= 18:
if → 如果（条件判断）   age → 变量   >= → 大于等于   18 → 数字   : → 冒号表示下面是代码块
8 print("你是成年人") # 注意这行前面有4个空格（缩进），表示属于 if 的内容

▎第二部分：if/else 二选一
条件成立走 if，否则走 else
10 # 2. if/else —— 二选一
注释：if/else 是最基础的"二选一"结构
11 score = 75
score → 变量（分数）   = → 赋值   75 → 数字
12
13 if score >= 60:
if → 条件判断开始   score → 变量   >= → 大于等于   60 → 及格线   : → 冒号
14 print("及格了！")
缩进4格 → 属于 if 块   条件为真时执行：打印及格信息
15 else:
else → 否则（当 if 条件为假时执行）   : → 冒号，后面接代码块
16 print("不及格，继续努力")
缩进4格 → 属于 else 块   条件为假时执行：打印提示

▎第三部分：if/elif/else 多选一
elif = else if，依次检查多个条件
18 # 3. if/elif/else —— 多选一（elif = else if 的缩写）
注释：elif 是 else if 的缩写，用于多个分支
19 score = 85
score → 变量重新赋值为 85，覆盖之前的 75
20
21 if score >= 90:
if → 第一个条件：score >= 90？  85 < 90，为假，跳过
22 print("优秀！A")
if 条件为假，此行跳过不执行
23 elif score >= 80:
elif → 否则如果（第二个条件）   score >= 80？  85 >= 80 为真，执行此分支
24 print("良好！B")
elif 条件为真，执行此行 → 输出"良好！B"，后续 elif/else 全部跳过
25 elif score >= 60:
elif → 第三个条件，因为第二个已匹配，此行不再检查
26 print("及格！C")
不执行（已有分支匹配）
27 else:
else → 所有条件都不满足时的兜底分支，此处不执行
28 print("不及格！D")
不执行（兜底 else 没有触发）

▎第四部分：比较运算符
== != > < >= <= 返回 True 或 False
30 # 4. 比较运算符
注释：介绍 Python 所有比较运算符
31 a = 10
a → 变量   = → 赋值   10 → 数字
32 b = 20
b → 变量   = → 赋值   20 → 数字
33 print(f"a={a}, b={b}")
print   f"..." → f-string，{a} 会替换成变量的值   输出：a=10, b=20
34 print(f"a == b ? {a == b}") # 等于（注意是两个等号）
== → 等于判断（双等号），与赋值 = 不同   10 == 20 为 False
35 print(f"a != b ? {a != b}") # 不等于
!= → 不等于   10 != 20 为 True   ! 在编程中常表示"非/否定"
36 print(f"a > b ? {a > b}") # 大于
> → 大于   10 > 20 为 False
37 print(f"a < b ? {a < b}") # 小于
< → 小于   10 < 20 为 True
38 print(f"a >= b ? {a >= b}") # 大于等于
>= → 大于等于   10 >= 20 为 False
39 print(f"a <= b ? {a <= b}") # 小于等于
<= → 小于等于   10 <= 20 为 True   六种运算符全部介绍完毕
40
▎第五部分：逻辑运算符
and / or / not 组合多个条件
41 # 5. 逻辑运算符 —— 组合多个条件
注释：用逻辑运算符把多个比较表达式组合在一起
42 age = 25
age → 变量重新赋值为 25
43 has_id = True # True 表示"是"，False 表示"否"
has_id → 布尔变量（有/没有身份证）   True → 真值，布尔类型只有 True / False 两种值
44
45 if age >= 18 and has_id: # and: 两个条件都满足
and → 且，两边条件都为真才为真   age >= 18（真）and has_id（真）→ 真，执行
46 print("可以进入酒吧")
两个条件都满足，输出"可以进入酒吧"
47
48 if age < 18 or has_id == False: # or: 至少一个条件满足
or → 或，至少一边为真即为真   age < 18（假）or has_id == False（假）→ 假，跳过
49 print("不能进入酒吧")
or 条件为假，此行不执行
50
51 if not (age < 18): # not: 取反，把 True 变 False
not → 取反，把括号内的结果反转   age < 18 为假，not 假 → 真，执行
52 print("你不是未成年人")
not 条件为真，执行：输出"你不是未成年人"

▎第六部分：实战小例子——简易计算器
用 if/elif/else 实现运算符分支
54 # 6. 实战小例子：简易计算器
注释：综合运用 if/elif/else 做一个简单计算器
55 num1 = 10
num1 → 第一个操作数，值为 10
56 num2 = 3
num2 → 第二个操作数，值为 3
57 operator = "+"
operator → 运算符变量   "+" → 字符串，存储用户选择的运算符号
58
59 if operator == "+":
if   == → 字符串相等判断   operator 是"+"？是，执行此分支
60 print(f"{num1} + {num2} = {num1 + num2}")
f-string 内 {num1 + num2} → 直接在字符串里计算 10+3=13，输出：10 + 3 = 13
61 elif operator == "-":
elif → 否则如果是减法，因上方已匹配，跳过
62 print(f"{num1} - {num2} = {num1 - num2}")
不执行（operator 不是"-"）
63 elif operator == "*":
elif → 否则如果是乘法，跳过
64 print(f"{num1} * {num2} = {num1 * num2}")
不执行（operator 不是"*"）
65 elif operator == "/":
elif → 否则如果是除法，跳过
66 if num2 != 0:
if 嵌套在 elif 内：先判断除数不为0，避免"除以0"错误
67 print(f"{num1} / {num2} = {num1 / num2}")
8空格缩进（嵌套两层） → 属于内层 if   / → 除法运算符
68 else:
内层 else（4空格缩进） → num2 == 0 时执行
69 print("错误：不能除以0！")
8空格缩进 → 内层 else 块，防止除以零崩溃
70 else:
外层 else（与第一个 if 对齐） → 运算符不在已知范围内时执行
71 print(f"不认识的运算符：{operator}")
兜底处理：用 f-string 把未知运算符打印出来

本课知识点

if
条件为真时执行代码块，冒号 : 后接缩进的语句
elif
else if 的缩写，用于多条件分支，按顺序逐一检查
比较运算符
== != > < >= <=，结果为 True 或 False
逻辑运算符
and or not 组合多个条件，构建复杂判断
