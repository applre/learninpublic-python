74 # 7. 实战：温度转换器
注释：第七部分标题，实际项目场景
75 def celsius_to_fahrenheit(c):
定义摄氏→华氏转换函数   c → 摄氏温度参数
76 return c * 9 / 5 + 32
转换公式：°C × 9/5 + 32   Python 按运算符优先级：先 * 后 / 再 +
77
空行
78 def fahrenheit_to_celsius(f):
定义华氏→摄氏转换函数   f → 华氏温度参数
79 return (f - 32) * 5 / 9
反向公式：(°F - 32) × 5/9   () → 括号保证先减 32
80
空行
81 temp_c = 37
变量 temp_c 存储摄氏温度 37（体温）
82 temp_f = celsius_to_fahrenheit(temp_c)
调用函数，传入 37 → 37×9/5+32 = 98.6   结果存入 temp_f
83 print(f"{temp_c}°C = {temp_f}°F")
输出：37°C = 98.6°F   °C °F 是文字，直接写在字符串里
84
空行
85 temp_f = 100
变量 temp_f 重新赋值为 100（华氏）
86 temp_c = fahrenheit_to_celsius(temp_f)
调用反向函数，传入 100 → (100-32)×5/9 ≈ 37.778   结果存入 temp_c
87 print(f"{temp_f}°F = {temp_c:.1f}°C") # :.1f 保留1位小数
:.1f → 格式化浮点数，保留1位小数   输出：100°F = 37.8°C

本课知识点

def
定义函数，把代码打包成工具。写一次，随时调用
参数
函数的输入，让工具更灵活。调用时按顺序传入具体值
return
返回结果，可继续参与计算。和 print 的区别：交回 vs 说出
默认参数
size="中杯" 不传就用默认值，传了就用新值
