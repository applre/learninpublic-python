Python 代码怎么读？

掌握三条规则，再复杂的代码你也能看懂

⬇️

规则1：从上往下

Python 从第1行开始，一行一行往下执行，就像读书一样

🔀

规则2：遇到 if 走岔路

碰到 if/elif/else，只走一条路，其他路全部跳过

📦

规则3：看缩进认主人

缩进（前面的空格）表示"属于谁"。同样缩进的代码是一组

示例1：从上往下，逐行执行

点击"下一步"按钮，跟着程序一行一行走

1 name = "jingyu" 1
2 age = 25 2
3 hobby = "coding" 3
4 print(f"我是{name}") 4
5 print(f"今年{age}岁") 5

执行顺序

1 把 "jingyu" 存进 name
2 把 25 存进 age
3 把 "coding" 存进 hobby
4 输出：我是jingyu
5 输出：今年25岁

就是这么简单，1→2→3→4→5，从上到下

示例2：if/else 只走一条路

score = 85 时，看看程序怎么"选路"。红色 = 被跳过的代码

1 score = 85 1
2
3 if score >= 90: 2
4 print("优秀！A") ✕
5 elif score >= 80: 3
6 print("良好！B") 4
7 elif score >= 60: ✕
8 print("及格！C") ✕
9 else: ✕
10 print("不及格！D") ✕
11
12 print("程序继续往下走") 5

执行顺序

1 score 设为 85
2 85 >= 90？不满足，跳过
3 85 >= 80？满足！进入这条路
4 输出：良好！B
5 剩下的 elif/else 全跳过，继续往下

关键：找到一条满足的路后，剩下的全部跳过！

示例3：缩进 = 层级关系

缩进越深，"嵌套"越深。就像文件夹里套文件夹

1 age = 20 1
2 has_ticket = True 2
3
4 if age >= 18: # 第一层判断 3
5 print("成年人") # ← 4个空格，属于外层 if 4
6 if has_ticket: # 第二层判断（嵌套） 5
7 print("有票，请进！") # ← 8个空格，属于内层 if 6
8 else: ✕
9 print("请先买票") ✕
10 else: ✕
11 print("未成年人不能进入") ✕
12
13 print("结束") 7

执行顺序

1 age = 20
2 has_ticket = True
3 20 >= 18？满足，进入
4 输出：成年人
5 has_ticket 是 True？满足
6 输出：有票，请进！
7 回到最外层，输出：结束

缩进就像俄罗斯套娃，一层套一层

记住三句话就够了

⬇️ 从上往下读，一行一行来

🔀 遇到 if，只走一条路，其余跳过

📦 看缩进判断谁属于谁
