22 a = 10

空行分隔
25 print(f"{a} + {b} = {a + b}") # 加法
+ → 加法运算符   {a + b} → 花括号内可以写表达式，Python 先计算结果再插入字符串   输出：10 + 3 = 13
26 print(f"{a} - {b} = {a - b}") # 减法
- → 减法运算符   输出：10 - 3 = 7
27 print(f"{a} * {b} = {a * b}") # 乘法
* → 乘法运算符（用星号而非 ×）   输出：10 * 3 = 30
28 print(f"{a} / {b} = {a / b}") # 除法（有小数）
/ → 除法，结果始终为浮点数   输出：10 / 3 = 3.3333…
29 print(f"{a} // {b} = {a // b}") # 整除（去掉小数）
// → 整除运算符，向下取整，去掉小数部分   输出：10 // 3 = 3
30 print(f"{a} % {b} = {a % b}") # 取余数
% → 取余（模运算），10 除以 3 余 1   输出：10 % 3 = 1
31 print(f"{a} ** {b} = {a ** b}") # 幂运算（10的3次方）
** → 幂运算符（乘方），10³ = 1000   输出：10 ** 3 = 1000

本课知识点

print()
内置函数，将括号内的内容输出到屏幕。可传入字符串、数字或变量。
变量赋值
用 = 给数据起名字。左边是变量名，右边是值，数字不加引号，文字加引号。
f-string
f"…{变量}…" 格式化字符串，花括号内可嵌入变量或表达式，比 + 拼接更方便。
运算符
+ 加   - 减   * 乘   / 除   // 整除   % 取余   ** 幂
