Agent Loop 到底在干什么？

用最通俗的方式，理解 AI Agent 的核心代码

先用生活类比理解

想象你是老板（用户），Claude 是你的助理，工具是助理手边的设备（计算器、搜索引擎等）

👤
你（老板）
"帮我查下天气"
→
🤖
Claude（助理）
我需要用天气工具
→
🔧
工具（天气API）
北京 25°C 晴
→
🤖
Claude（看到结果）
北京今天25度，晴天
↻ 如果还需要用工具，就再循环一圈（比如"还需要查明天的"）

程序执行流程

开始：带着对话记录调用 Claude
↓
Claude 思考并返回回复（response）
↓
把 Claude 的回复追加到对话记录
↓
判断：Claude 要用工具吗？
stop_reason == "tool_use" ?
不用工具
↓
要用工具
↓
结束，返回最终回答给用户
执行工具，收集结果
↓
把工具结果追加到对话记录
↑ 回到顶部，再次调用 Claude

逐行代码解析

点击"下一步"逐段理解每块代码在做什么

1 def agent_loop(messages):
2 while True:
3 response = client.messages.create(
4 model=MODEL, system=SYSTEM,
5 messages=messages, tools=TOOLS,
6 )
7 messages.append({"role": "assistant",
8 "content": response.content})
9
10 if response.stop_reason != "tool_use":
11 return
12
13 results = []
14 for block in response.content:
15 if block.type == "tool_use":
16 output = TOOL_HANDLERS[block.name](
17 **block.input)
18 results.append({
19 "type": "tool_result",
20 "tool_use_id": block.id,
21 "content": output,
22 })
23 messages.append({"role": "user",
24 "content": results})

代码解读

1定义函数
messages 是对话记录列表，就像微信聊天记录，记录了你和 AI 说过的所有话
2无限循环
while True = 一直循环。为什么？因为 AI 可能需要连续使用多个工具才能完成任务，每用一次工具就是一轮循环
3调用 Claude
把对话记录和可用工具一起发给 Claude API。Claude 看完后会回复——可能是文字回答，也可能是"我要用某个工具"
4记录 AI 的回复
把 Claude 的回复追加到对话记录中，角色标记为 assistant。就像把助理说的话记到聊天记录里
5判断：要不要用工具？
如果 stop_reason 不是 "tool_use"，说明 Claude 已经给出了最终答案，直接 return 结束循环。否则继续往下走
6执行工具
遍历 Claude 回复中的每个工具调用请求，用 TOOL_HANDLERS（工具函数表）找到对应的函数并执行，把结果收集起来
7把工具结果喂回去
把工具执行结果追加到对话记录中，角色标记为 user（API 规定工具结果要以 user 身份发送）。然后回到 while 循环顶部，再次调用 Claude

模拟一次完整的 Agent Loop

假设用户问"北京和上海今天谁更热？"，AI 需要调用两次天气工具

👤
用户 (messages 初始内容)
北京和上海今天谁更热？
—— 第1轮循环 ——
🤖
Claude 回复 (stop_reason = "tool_use")
我需要查两个城市的天气。先查北京：
→ 调用工具: get_weather("北京")
🔧
工具执行结果 (追加到 messages)
北京：32°C，晴
—— 第2轮循环 ——
🤖
Claude 回复 (stop_reason = "tool_use")
北京的结果拿到了，再查上海：
→ 调用工具: get_weather("上海")
🔧
工具执行结果 (追加到 messages)
上海：28°C，多云
—— 第3轮循环 ——
🤖
Claude 回复 (stop_reason = "end_turn") ✅ 循环结束！
北京今天 32°C，上海 28°C。北京更热，比上海高 4 度。

一句话总结

Agent Loop = 不断循环"问 AI → 执行工具 → 把结果告诉 AI"，直到 AI 不再需要工具为止

💬

messages 对话记录

所有对话的"记忆"，每轮都往里追加，这样 Claude 能看到完整上下文

🔁

while True 循环

一直转，直到 Claude 说"我不需要工具了"才 return 退出

🔧

工具调用

Claude 不能直接上网/算数，但它可以告诉你"我要用XX工具"，由代码帮它执行

🔑

stop_reason

Claude 的"信号灯"。tool_use = 还要用工具，其他 = 说完了
