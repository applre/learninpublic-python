▎1. for 循环 — 遍历列表
依次取出列表中每个元素
4 # 1. for 循环 —— 遍历一个列表
#注释 → 段落标题注释
5 # 列表就是用 [] 括起来的一组数据
#注释 → 解释列表语法，[] 是列表的标志
6 fruits = ["苹果", "香蕉", "橘子"]
变量 fruits → 存了一个列表，列表里有 3 个字符串元素，用逗号分隔，外面套 []
8 for fruit in fruits:
for fruit in fruits → 每次循环从 fruits 取一个元素，放入临时变量 fruit，共循环 3 次；末尾 : 必须写
9 print(f"我喜欢吃{fruit}")
print → 缩进 4 格表示属于循环体；f"..." 中的 {fruit} 自动替换为当前元素值
11 print("---")
print → 无缩进，在循环结束后执行一次，打印分隔线
▎2. range() — 生成数字序列
range 控制循环次数
13 # 2. range() —— 生成一串数字
#注释 → 段落标题
14 # range(5) 生成 0,1,2,3,4（从0开始，不包含5）
#注释 → 重要规则：range 从 0 开始，终点值不包含在内
15 for i in range(5):
range(5) → 生成序列 0,1,2,3,4；i 依次取每个值，循环 5 次
16 print(f"第{i}次循环")
print → 循环体，输出 "第0次循环" … "第4次循环"，共 5 行
18 print("---")
print → 循环结束后打印分隔线
20 # range 还可以指定起点和步长
#注释 → range 的进阶用法
21 # range(起点, 终点, 步长)
#注释 → 三参数格式：起点（含）、终点（不含）、步长（每次加多少）
22 for i in range(1, 10, 2): # 1,3,5,7,9
range(1,10,2) → 从 1 开始，每次 +2，到 10 前停止，得到 1,3,5,7,9
23 print(i, end=" ") # end=" " 表示不换行，用空格隔开
end=" " → print 默认在末尾加换行，设为空格后不换行，所有数字打印在同一行，空格分隔
24 print() # 换行
print() → 无参数调用只输出一个换行符，让光标回到下一行
26 print("---")
print → 打印分隔线
▎3. while 循环 — 条件控制
条件为真就一直重复
28 # 3. while 循环 —— 条件满足就一直做
#注释 → 段落标题
29 # for 是"数着做"，while 是"满足条件就继续做"
#注释 → 对比记忆：for 知道次数，while 看条件决定
30 count = 0
变量 count → 初始化计数器为 0，while 循环需要手动管理计数变量
32 while count < 3:
while → 每次执行前检查条件 count < 3；为 True 就进入循环体，为 False 就停止
33 print(f"count = {count}")
print → 打印当前 count 的值，缩进表示属于 while 循环体
34 count = count + 1 # 每次 +1，不然会无限循环！
count +1 → 关键！必须修改条件变量，否则条件永远为 True，程序卡死（无限循环）
36 print(f"循环结束，count = {count}")
print → while 结束后执行，此时 count 已是 3（不满足 < 3 才退出）
38 print("---")
print → 打印分隔线
▎4. break — 提前退出循环
找到目标立即停下
40 # 4. break —— 提前退出循环
#注释 → 段落标题
41 # 找到目标就停下来，不用把剩下的都看完
#注释 → break 的使用场景：搜索到结果后无需继续遍历
42 numbers = [1, 5, 8, 3, 12, 7]
变量 numbers → 一个整数列表，包含 6 个元素，其中 12 是第一个大于 10 的数
44 for num in numbers:
for num in numbers → 遍历 numbers 列表，num 依次为 1, 5, 8, 3, 12, 7
45 print(f"检查 {num}...")
print → 每次循环先打印"正在检查"，显示遍历过程
46 if num > 10:
if → 判断当前元素是否大于 10，满足时执行缩进块
47 print(f"找到了大于10的数：{num}！停止搜索")
print → 打印找到的目标值，双重缩进（属于 if 块，if 属于 for 块）
48 break # 立刻跳出循环
break → 立即终止整个 for 循环，后续的 7 不再检查，跳到循环后的代码
50 print("---")
print → break 退出循环后从这里继续执行
▎5. continue — 跳过本次
跳过当前轮次，继续下一次
52 # 5. continue —— 跳过这一次，继续下一次
#注释 → continue 与 break 的区别：break 结束循环，continue 只跳过本次
53 for i in range(1, 6):
range(1,6) → 生成 1,2,3,4,5，i 依次取这 5 个值
54 if i == 3:
if → 当 i 等于 3 时触发特殊处理
55 print(f"跳过 {i}")
print → 打印"跳过 3"，说明进入了跳过逻辑
56 continue # 跳过下面的 print，直接进入下一次循环
continue → 跳过本轮循环剩余的代码（下面的 print），直接回到 for 取下一个值
57 print(f"处理 {i}")
print → 只在 i != 3 时执行，输出 "处理 1" "处理 2" "处理 4" "处理 5"（跳过了3）
59 print("---")
print → 打印分隔线
▎6. 实战：累加求和
计算 1+2+3+…+100
61 # 6. 实战：累加求和
#注释 → 段落标题
62 # 计算 1+2+3+...+100
#注释 → 经典编程练习题
63 total = 0
变量 total → 累加器，初始为 0，每次循环往里加一个数
64 for i in range(1, 101): # 1到100
range(1,101) → 生成 1 到 100（注意：终点写 101 才能包含 100），i 依次取每个值
65 total = total + i
total += i → 每次把 i 加入 total，循环 100 次后 total 就是 1+2+…+100 的结果
67 print(f"1+2+3+...+100 = {total}")
print → 循环结束后输出结果，答案是 5050（高斯求和公式验证）
▎7. 实战：九九乘法表
循环嵌套的经典示例
69 # 7. 实战：九九乘法表
#注释 → 段落标题
70 for i in range(1, 10):
外层 for → i 从 1 到 9，控制乘法表的行（每行对应一个被乘数）
71 for j in range(1, i + 1): # 循环里套循环（嵌套）
内层 for → j 从 1 到 i，控制每行的列；range(1, i+1) 使每行列数随 i 增加（三角形）
72 print(f"{j}×{i}={i*j}", end="\t") # \t 是制表符，对齐用
print end="\t" → 打印 "j×i=结果"，\t 是制表符（Tab），让每列对齐；不换行，同行继续
73 print() # 每行结束换行
print() → 内层循环结束后换行，属于外层 for 的循环体（一行结束，换下一行）
▎练习：enumerate 同时获取序号
用 enumerate 打印编号列表
75 # === 练习 ===
#注释 → 练习区域标题
76 # 用 for 循环打印你最喜欢的5部电影/动漫/游戏
#注释 → 练习题说明，可以修改列表内容
77 my_favorites = ["在这里写第1个", "第2个", "第3个", "第4个", "第5个"]
变量 my_favorites → 包含 5 个占位字符串的列表，把引号里的内容换成你喜欢的即可
79 print("\n我的最爱TOP5：")
print → \n 是换行转义字符，先输出空行再输出标题
80 for i, item in enumerate(my_favorites, 1): # enumerate 同时拿到序号和内容
enumerate → 同时返回序号和元素，i 是序号，item 是内容；第二个参数 1 表示从 1 开始计数（默认 0）
81 print(f" 第{i}名：{item}")
print → 输出 "第1名：…" 到 "第5名：…"，两个空格缩进让输出更美观

while

break / continue
提前退出 / 跳过本次
