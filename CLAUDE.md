# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

jingyu 是一名产品经理，正在从零学习编程（Python 起步），并将学习过程公开分享到微信公众号和小红书。本仓库既是学习代码的工作区，也是内容创作的素材库。

目标受众：和 jingyu 一样的非技术背景人群（产品、运营、设计等），内容风格需通俗易懂、有趣可视化。

## Structure

- `python-basics/` — Sequential Python lesson files (`01_hello.py`, `02_if_else.py`, `03_loops.py`, etc.) and HTML visual guides
- `.obsidian/` — Obsidian vault configuration (do not modify)

## Running Code

```bash
python3 python-basics/<filename>.py
```

## Conventions

- Lesson files are numbered sequentially (`01_`, `02_`, `03_`, ...)
- Each lesson file is self-contained with comments in Chinese explaining concepts
- Code comments and explanations should be in Chinese (中文)
- Each lesson ends with a `# === 练习 ===` section for hands-on practice
- HTML files are used for interactive visual explanations of programming concepts
